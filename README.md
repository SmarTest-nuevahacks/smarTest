# smarTest
Smartest is a webapp focused on creating cheat-free online tests and exams for remote learning.

Installation steps:
[Linux]
1. `chmod +x run.sh`
2. `./run.sh`

[MacOS]
1. `chmod +x run.sh`
2. `sh run.sh`

The experience is optimized under Ubuntu 18.04 with Mozilla Firefox and Python 3.7.
